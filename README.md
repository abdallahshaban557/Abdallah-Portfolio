# Abdallah's Portfolio

This is the source code for my portfolio website. It is built using [Astro](https://astro.build/) and [Tailwind CSS](https://tailwindcss.com/).

## Features

- Dark mode
- Responsive design
- SEO optimized
- Blog with frontmatter (title, description, author, date, image, tags) and RSS feed, sitemap and robots.txt
- 404 error page
